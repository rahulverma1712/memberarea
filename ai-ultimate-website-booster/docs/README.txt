AI Ultimate Website Booster
===========================

A premium WordPress plugin by Swastik Infotech that delivers AI content generation, rich schema markup, premium popup templates, and website health tools in one polished admin interface.

Core Features:
- Real OpenAI integration with custom endpoint support.
- AI article draft generation with tone selection.
- Premium popup templates with exit intent, scroll, and timed triggers.
- SEO JSON-LD markup for single posts.
- Website health checks and inactive plugin notifications.
- Secure AJAX and REST endpoints.
- Custom action log table for monitoring actions.

Package Contents:
- ai-ultimate-website-booster.php
- admin/ (dashboard and settings UI)
- includes/ (AI integration, popup engine, schema, health, REST)
- assets/ (styles and JavaScript)
- templates/ (premium popup templates)
- docs/ (installation, marketing copy, screenshot guides)
