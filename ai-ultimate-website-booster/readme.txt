AI Ultimate Website Booster is a high-converting WordPress plugin by Swastik Infotech that combines AI content generation, premium popup templates, rich SEO schema, and website health checks in one professional admin experience.

== Description ==

This plugin is built for agencies, marketers, and store owners who want a complete WordPress growth toolkit. It supports real AI integration with OpenAI, secure administration, REST and AJAX workflows, and flexible popup campaigns with premium templates.

== Features ==

* AI content generation using OpenAI or custom endpoint.
* Premium popup templates with exit intent, scroll, and delay triggers.
* JSON-LD schema injection for single posts.
* Admin dashboard with tabbed navigation and modern UI.
* Website health audit and inactive plugin notices.
* Secure nonce validation and capability checks.
* Custom logging table for action tracking.
* Fully documented installation and marketing-ready packaging.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/ai-ultimate-website-booster`.
2. Activate the plugin from the WordPress Plugins screen.
3. Go to the AI Booster admin menu and configure API, popup, and schema settings.

== Frequently Asked Questions ==

= How do I enable real AI generation? =
Enter your OpenAI API key in Settings and choose the model. The plugin will use the key to generate content live.

= Can I customize popup text? =
Yes. The Popups tab includes headline, message, button text, and CTA URL fields for each template.

= Is the plugin safe for CodeCanyon review? =
Yes. It includes secure validation, sanitized inputs, and clean WordPress coding standards.
